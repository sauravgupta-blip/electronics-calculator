# 🔌 Electronics Calculator

A beginner-friendly Python project for basic electronics calculations.

## 🚀 Features
- Ohm’s Law Calculator (V = I × R)
- Power Calculator (P = V × I)
- Resistor Color Code Calculator

## 🧪 Example
Input:
red, violet, yellow  
Output:
270000 Ohms

## ▶️ How to Run
```bash
python3 calculator.py
